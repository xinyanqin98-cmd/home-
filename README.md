# 平面图房间识别训练台

这是一个用于演示和采集训练样本的网页原型。用户上传平面图后，页面会在浏览器本地估算图中有几个房间，并允许人工修正结果。

## 使用方式

直接打开 `index.html`，或使用本地预览服务：

```powershell
node server.js
```

然后访问：

```text
http://127.0.0.1:4173/
```

## 当前功能

- 上传 JPG、PNG、WEBP 平面图
- 用本地图像算法估算疑似房间数量
- 在图纸上标出疑似房间区域
- 支持人工修正房间数
- 将“模型估算 + 人工标注”记录为训练样本
- 导出 JSON 训练数据

## 已完成需求

- 将原来的家装概念方案页面，改成“平面图房间识别训练台”
- 用户可以上传平面图，让系统初步判断图中有几个房间
- 页面会显示当前估算房间数和识别可信度
- 用户可以手动修正房间数量，作为准确标注
- 用户可以把每次上传的平面图、系统估算结果、人工标注结果保存为训练样本
- 用户可以导出训练样本 JSON，后续用于训练真正的视觉 AI 模型
- 修复原页面中文乱码问题，统一改为 UTF-8 中文文案

## 说明

当前版本不是深度学习模型，也不会在浏览器里真正训练 AI。它更像第一步：先收集带标签的数据。等样本数量足够后，可以用这些数据训练更稳定的视觉模型，例如目标检测或图像分割模型。
