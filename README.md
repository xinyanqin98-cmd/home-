# 宅构概念设计

一个面向家装概念方案的网页原型。用户上传一楼平面图后，页面会把图纸作为空间依据，并围绕 Kitchen、Dining、Family、Lounge 四个空间生成设计说明和实景概念渲染卡片。

## 使用方式

直接打开 `index.html`，或使用本地预览服务：

```powershell
node server.js
```

然后访问：

```text
http://127.0.0.1:4173/
```

## 当前功能

- 上传平面图并固定预览
- 记录一楼四空间约束：Kitchen、Dining、Family、Lounge
- 输入或快速选择装修风格
- 生成每个空间的布局、材质、灯光和渲染提示词
- 用前端视觉卡片模拟室内实景概念渲染
